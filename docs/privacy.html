Privacy Policy

Instagram DM Mini Window Chrome Extension

1. Overview

This extension ("Instagram DM Mini Window") only opens and closes Instagram Direct Messages in a small popup window. It does not collect, store, sell, or share personal information.

2. Data We Collect

This extension does not collect:

  • Personally identifiable information (name, email, phone number, etc.)
  • Sensitive information (message content, passwords, etc.)
  • Browsing history, location data, or payment information

3. Permissions and Purpose

  • tabs, windows: Used to open, close, and position the Instagram DM popup window.
  • host_permissions (instagram.com): Used only to open Instagram DM URLs.

4. Data Storage and Sharing

  • No user data is stored by this extension.
  • No user data is transmitted to external servers.
  • No user data is shared with third parties.

5. User Rights

Since this extension does not process personal data, there is no personal data to access, correct, or delete. You can stop using the service at any time by uninstalling the extension.

6. Children's Privacy

This extension does not knowingly collect or process children's personal information.

7. Policy Updates

This Privacy Policy may be updated as needed, and any updates will be posted on this page.

8. Contact

For questions about this Privacy Policy, please contact us via GitHub issues or the developer contact listed in the Chrome Web Store.

Effective date: 2026-03-12

Instagram DM Mini Window Chrome Extension

1. 개요

본 확장 프로그램("Instagram DM Mini Window")은 인스타그램 DM 페이지를 작은 팝업 창으로 열고 닫는 기능만 제공합니다. 사용자의 개인정보를 수집, 저장, 판매 또는 제3자에게 제공하지 않습니다.

2. 수집하는 정보

본 확장 프로그램은 다음 정보를 수집하지 않습니다.

  • 이름, 이메일, 전화번호 등 개인식별정보
  • 메시지 내용, 계정 비밀번호 등 민감한 정보
  • 브라우징 기록, 위치 정보, 결제 정보

3. 권한 사용 목적

  • tabs, windows: 인스타그램 DM 창을 열고, 닫고, 위치를 제어하기 위해 사용합니다.
  • host_permissions (instagram.com): 인스타그램 DM URL을 열기 위해 사용합니다.

4. 데이터 보관 및 제3자 제공

  • 본 확장 프로그램은 사용자 데이터를 저장하지 않습니다.
  • 외부 서버로 사용자 데이터를 전송하지 않습니다.
  • 제3자에게 사용자 데이터를 제공하지 않습니다.

5. 사용자 권리

본 확장 프로그램은 개인정보를 처리하지 않으므로 별도의 열람/정정/삭제 요청 대상 데이터가 없습니다. 확장 프로그램 삭제를 통해 언제든 사용을 중단할 수 있습니다.

6. 아동의 개인정보

본 확장 프로그램은 아동의 개인정보를 별도로 수집하거나 처리하지 않습니다.

7. 정책 변경

본 개인정보처리방침은 필요 시 업데이트될 수 있으며, 변경 시 본 페이지를 통해 공지합니다.

8. 문의

개인정보처리방침 관련 문의는 GitHub 저장소 이슈 또는 웹스토어 개발자 연락처를 통해 접수해 주세요.
